# AIYL Taxi — Описание проекта

## Что это?

Мобильное приложение для вызова такси в селе. Простая и удобная замена телефонным звонкам диспетчеру. Пассажир нажимает одну кнопку — система автоматически находит ближайшего свободного водителя.

---

## Роли в системе

### Пассажир (Клиент)

- Входит в приложение по номеру телефона (приходит SMS-код из 4 цифр)
- Видит свое местоположение на карте
- Вызывает такси одной кнопкой — цена показывается сразу
- В реальном времени видит статус заказа:
  - "Ищем водителя..."
  - "Водитель принял заказ" — отображается имя, машина, номер телефона
  - "Водитель прибыл" — приходит уведомление
  - "Поездка завершена" — показывается итоговая цена
- Может отменить заказ до начала поездки
- Может позвонить водителю прямо из приложения (одним нажатием)
- Видит историю своих поездок

### Водитель

- Входит по номеру телефона и паролю (аккаунт создает администратор)
- Включает/выключает режим "На линии"
- Когда на линии — его GPS-координаты передаются в систему
- Получает предложения о новых заказах с обратным отсчетом (10 секунд на решение):
  - Расстояние до пассажира
  - Цена поездки
  - Адрес подачи
- Принимает или отклоняет заказ
- После принятия:
  - Видит пассажира на карте
  - Может открыть навигацию к точке подачи
  - Может позвонить пассажиру
  - Нажимает "Прибыл" по приезду
  - Нажимает "Завершить поездку" после доставки
- Видит свою статистику: заработок за сегодня, неделю, месяц

### Администратор

- Входит через веб-браузер (компьютер или телефон)
- Видит панель управления:
  - Количество активных заказов
  - Сколько водителей на линии
  - Выручка за сегодня
  - Общее количество поездок
- Управляет водителями:
  - Создает аккаунты (имя, телефон, пароль, марка и номер машины)
  - Редактирует данные
  - Удаляет водителей
- Просматривает все заказы с фильтрацией по статусу

## Тарифы

| Время | Цена |
|-------|------|
| Дневной (07:00 — 21:00) | **80 сом** |
| Ночной (21:00 — 07:00) | **120 сом** |

Цена фиксированная — пассажир видит её до вызова такси.

---

## Как работает поиск водителя

1. Пассажир нажимает "Вызвать такси"
2. Система находит всех водителей "на линии" и сортирует по расстоянию
3. Ближайшему водителю отправляется предложение
4. У водителя есть **10 секунд** на ответ
5. Если водитель отклонил или не ответил — предложение автоматически переходит к следующему ближайшему
6. Если свободных водителей нет — пассажир получает сообщение "Нет свободных водителей"

---

## Уведомления

Приложение отправляет push-уведомления даже когда свернуто:

**Пассажиру:**
- Водитель принял заказ
- Водитель прибыл
- Поездка завершена

**Водителю:**
- Новый заказ рядом

## Технические детали

- **Мобильное приложение**: работает на iOS и Android (одно приложение для пассажиров и водителей)
- **Админ-панель**: открывается в любом браузере
- **Карты**: встроенные нативные карты (Apple Maps / Google Maps)
- **GPS**: автоматическое определение местоположения
- **Обновления в реальном времени**: статус заказа и положение водителя обновляются мгновенно
- **Установка**: приложение устанавливается как обычное приложение (APK для Android, TestFlight для iOS)
- **Обновления приложения**: можно обновлять без повторной установки (OTA-обновления)
