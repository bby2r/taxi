Alif Taxi — современное такси для жителей Таласской области

• Заказ такси внутри села — фиксированная цена, без сюрпризов

• Реальная карта с местоположением водителя и расчётным временем
• Уведомление когда водитель прибыл — даже если приложение свёрнуто
• Возможность связаться с водителем одним тапом
• Оценка водителя после поездки — помогите нам поддерживать качество
• История ваших поездок с детализацией по каждой
• Удобный вход по номеру телефона — без паролей, через SMS-код

НАШИ ВОДИТЕЛИ
Местные водители, которые знают каждую улицу в Таласе, Кировке
и Покровке. У каждого водителя есть проверенный профиль с фото,
маркой автомобиля и рейтингом от пассажиров.

ПРОЗРАЧНОЕ ЦЕНООБРАЗОВАНИЕ
Цена поездки известна до её заказа. Никаких скрытых надбавок и

процент.

ПОДДЕРЖКА

момент заказа. Подробности — в политике конфиденциальности
(ссылка в Профиле).

Alif Taxi — независимый сервис, разработанный для местных жителей.
