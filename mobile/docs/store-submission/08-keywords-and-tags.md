# Keywords — Apple App Store

App Store позволяет до **100 символов keywords** через запятую.
**Keywords НЕ повторяй слова из названия/subtitle** — Apple их и так индексирует.

# Google Play — Tags / Categories

Google Play не имеет ручного поля keywords. Поисковая индексация —
через **Title + Short description + Long description**. Главное чтоб
там органично встречались:
- такси, taxi
- Талас, Talas, Talas Region
- Кировка, Покровка
- Кыргызстан, Kyrgyzstan
- межсёлами, intercity

Это уже есть в descriptions (см. файлы 02 и 03).
